# กำหนดรูปแบบการค้นหาข้อมูล (ในเมนูรายวันต่างๆ)

SML สามารถกำหนดรูปแบบการค้นหาข้อมูล (ในเมนูรายวันต่างๆ)
เพื่อให้ออกมาในเมนูฝั่งซ้ายมือเพื่อสะดวกในการดูข้อมูล การค้นหา
