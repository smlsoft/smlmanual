# วิธีรับเงินจากการขาย POS

ระบบขาย POS สามารถทำการรับเงินได้หลายรูปแบบ ทั้งเงินสด บัตรเครดิต คูปอง
และการนำใบรับเงินล่วงหน้า หรือมัดจำมาตัดชำระจากการขายหน้าจอ Point of sale
(POS)
