# เปลี่ยนรหัสผ่าน

ในการแก้ไขผู้ดูแลระบบนั้นสามารถแก้ไขได้ 2 ส่วนคือ ในส่วนของ
**“การแก้ไขข้อมูลผู้ใช้”** และในส่วนของ**“การเปลี่ยนรหัสผ่าน”**
ผู้ดูแลระบบสามารถแก้ไขรหัสผ่านของตนได้จากในหัวข้อนี้ด้วย
