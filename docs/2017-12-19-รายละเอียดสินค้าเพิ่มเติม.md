# รายละเอียดสินค้าเพิ่มเติม

สำหรับหัวข้อนี้ ใช้สำหรับกำหนดค่าการทำงาน ให้แก่สินค้าแต่ละรายการ
รวมถึงการออกแบบข้อมูลให้สามารถนำมาใช้งานได้ตามความต้องการของแต่ละประเภทธุรกิจได้ครบวางจร
