# ตั้งค่าอนุญาติสินค้าคงเหลือติดลบได้

อาจมีบางสินค้าที่ยังนับสินคึ้าไม่เสร้จ ระบบสามารถ
กำหนดได้ว่าจะให้รายการไหนสามารถติดลบได้ และรายการใดที่นับแล้ว
ไม่ให้ติดลบ(ต้องหาสาเหตุ)
