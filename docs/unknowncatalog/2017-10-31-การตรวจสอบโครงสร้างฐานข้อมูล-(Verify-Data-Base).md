# การตรวจสอบโครงสร้างฐานข้อมูล (Verify Data Base)

การตรวจสอบโครงสร้างฐานข้อมูล **(Verify Data Base)**
เมื่อฐานข้อมูลมีการเปลี่ยนแปลงโครงสร้างจากการupdate โปรแกรม หรือ
จากการนำเข้าข้อมูล แนะนำให้ทำการ verify โดยมีวิธีการง่ายๆ ดังนี้
