# ขอเบิกสินค้า/วัตถุดิบ

ระบบการเบิกและขอเบิกสินค้าและวัตถุดิบ สามารถสร้างการทำงานให้สอดคล้องได้หลากลาย
ไม่ว่าจะเบิกใช้ภายใน เบิกไปผลิต หรือ เบิกไปเป้นค่าใช้จ่ายต่างๆของกิจการ
