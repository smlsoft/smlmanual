# บันทึกลูกหนี้ยกมา

สำหรับบันทึกยอดกมาของเจ้าหนี้การค้า หรือเจ้าหนี้กิจการประเภทอื่นๆ
โดยแบ่งเป็น**ยอดลูกหนี้ยกมา** **ยอดเพิ่มหนี้ลูกหนี้ยกมา และ
ยอดลดหนี้ลูกหนี้ยกมา**
